# FenLight AM (Android TV)

This repository hosts a highly optimized, uniquely named version of the Fen Light Kodi addon (`plugin.video.fenlight.am`), tailored specifically for low-resource environments such as older Android TVs and Firesticks. Memory overhead has been dramatically reduced, and CPU cycles have been simplified to ensure swift directory loading capabilities without clashing with the original fork!

## User Installation Guide

To install this addon on Kodi, follow these steps to add the repository and install the optimized FenLight AM addon:

### 1. Add the Repository Source
1. Open up Kodi and navigate to **Settings** (the gear icon at the top left).
2. Go to **File manager** -> **Add source**.
3. Click on `<None>` and enter the following URL precisely:
   `https://vivekkushwaha46.github.io/fenlight-am/repo/`
4. Name the media source (e.g., `FenLight AM`) and click **OK**.

### 2. Install the Repository ZIP
1. Go back to the Kodi home screen and select **Add-ons**.
2. Click the **Add-on browser** box icon at the top left.
3. Select **Install from zip file**. (If prompted, briefly enable "Unknown sources" in your Kodi Settings).
4. Choose the source you named earlier (`FenLight AM`).
5. Open the `plugin.video.fenlight.am/` folder and select the newest zip file.
6. Wait for the "Add-on installed" pop-up notification.

### 3. Verify the Add-on
1. The Add-on should directly appear within your main screen context items under **My add-ons > Video add-ons** as **FenLight AM**.
2. Ensure you apply any account configurations needed (e.g. Trakt / Debrid authorizations) within the app settings.
